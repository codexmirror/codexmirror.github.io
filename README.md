# Grundstücks-Realitätscheck

Statisches Vanilla-Webtool für eine schnelle Ersteinschätzung zur Plausibilität eines Bauvorhabens auf einem Grundstück.

## Start

Einfach `index.html` in einem Browser öffnen oder das Verzeichnis über GitHub Pages ausliefern.

## Einschränkungen

Keine Rechtsberatung. Das Tool nutzt feste Heuristiken und ersetzt keine Auskunft von Bauamt/Fachleuten.
